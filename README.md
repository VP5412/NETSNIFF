**NetSniff Pro v2.0** is an advanced, high-performance packet analysis suite built for modern security research. It bridges the gap between raw hardware capturing and intuitive data visualization, featuring a powerful **Python Engine** for real-time traffic dissection and a **State-of-the-Art Web Interface** for forensic inspection.

---

## 🚀 Key Features


### 🌐 Interactive Forensic UI (`netsniff_pro_cli.html`)
- **Zero-Install Analysis**: Run a full network analyzer directly in your browser.
- **Hex/ASCII Inspector**: Detailed breakdown of raw packet data with protocol-aware highlighting.
- **Intelligent Analytics**: Visual traffic intensity matrix and anomaly auditing logs.
- **Responsive Architecture**: Engineered to provide a consistent "Dark Mode" terminal experience across all devices.

---

## 🛠️ Technology Stack

- **CLI Core**: Python 3.8+ (`socket` raw-sockets, `struct` binary parsing).
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), and ES6 JavaScript.
- **Heuristics**: Custom time-window algorithms for threat pattern recognition.

### Exploring the Web Interface
